# Laboration 1 - Grunderna i TypeScript

## Beskrivning
I denna laboration skapas en webbplats som hanterar kurser med kurskod, kursnamn, progression och syllabus(länk).
Webbplatsen använder sig av Typescript där bland annat localstorage och DOM-manipulation används.

## Publicerad webbplats
Webbplatsen publiceras automatiskt via **GitHub Pages**.
